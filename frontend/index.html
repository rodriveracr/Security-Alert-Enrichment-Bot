Intelligence Sources

VirusTotal

Aggregates data from 90+ antivirus engines and malware databases

What we get:

  • Detected URLs: Number of malicious URLs associated
  • Reputation: Overall threat reputation score

Logic: Identifies if the indicator has been flagged as malicious by multiple security vendors

AbuseIPDB

Crowdsourced database of reported malicious IP addresses

What we get:

  • Abuse Score: 0-100 confidence score of malicious activity
  • Reports: Number of community reports filed against the IP

Logic: Determines risk level based on community submissions and abuse patterns

Shodan

Internet-wide port scanner and service discovery engine

What we get:

  • Open Ports: List of publicly accessible ports
  • Organization: ISP or organization owning the IP

Logic: Maps network exposure and identifies potential attack vectors through open ports

How to Use & Field Descriptions

About This Tool

This application enriches security alerts (IPs or domains) by querying multiple external threat intelligence sources and providing a clear summary for security analysts.

Field Descriptions

  • Type: Whether the indicator is an IP Address or a Domain
  • Value: The indicator being queried (e.g., 1.1.1.1)
  • VirusTotal - Detected URLs: Number of malicious URLs related to the indicator
  • VirusTotal - Reputation: Reputation score from VirusTotal (higher = more suspicious)
  • AbuseIPDB - Score: Abuse confidence score (0-100), higher indicates greater abuse probability
  • AbuseIPDB - Reports: Total accumulated reports in AbuseIPDB for the IP
  • Shodan - Open Ports: Open ports detected by Shodan on the IP (if applicable)
  • Shodan - Organization: IP owner organization according to Shodan
